 Wildlife Photography Website

 📸 Overview
This project is a **Wildlife Photography website** designed to showcase nature and photography content with a clean and responsive UI.
It includes multiple pages such as navigation, login/signup, and policy sections.

✨ Features
* 🧭 Custom Navigation Bar (Home, About, Info, Contact)
* 🖼️ Hero section with wildlife photography banner
* 🔐 Login Page
* 📝 Signup Page
* 📄 Terms & Conditions Page
* 📱 Responsive layout using Bootstrap

---

🛠️ Tech Stack
* HTML5
* CSS3
* Bootstrap

 📁 Project Structure
project-folder/
│── nav.html          # Main landing page with hero image
│── index.html        # Login page
│── insta2.html       # Signup page
│── policy.html       # Terms & conditions
│── nav.css           # Styling for navbar & hero section
│── insta.css         # Styling for forms
│── wildlife-photography.jpeg  # Background image
 
⚙️ How to Run
1. Download or clone the project
2. Open the folder in VS Code
3. Run using Live Server
   OR open `nav.html` in your brower

⚠️ Notes
* This is a **frontend-only project**
* Some pages use different styles (can be unified)
* Can be extended into a full photography portfolio platform
